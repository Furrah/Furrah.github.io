Installing CMake on macOS and adding to path

Cmake can be installed using

brew install cmake

if you have the homebrew package manager installed. If not, go to CMake and download the latest version.

once installed add to path using the command

sudo /Applications/CMake.app/Contents/bin/cmake-gui --install=/usr/local/bin

Done!
